# Supervised Learning

Supervised learning = training a model on data where **every row already has the correct answer (label)**, so the model learns the mapping from **input → output**.

```
Input (Features)  →  Model  →  Output (Label)
```

---

## Example Dataset — 5000 Students

| IQ  | CGPA | Placement (Y/N) |
|-----|------|------------------|
| 87  | 7.1  | Y                |
| 111 | 8.9  | Y                |
| 75  | 6.3  | N                |

- **Input (Features)** → `IQ`, `CGPA`
- **Output (Label)** → `Placement (Y/N)`

The model looks at thousands of rows like this and learns the pattern connecting IQ/CGPA to whether a student got placed.

**Query:** Given a new student's IQ and CGPA → predict `Y` or `N`?
→ This is a **Classification** problem, because the output is a category (`Y` / `N`), not a number.

---

## Second Example — Predicting Package

| IQ  | CGPA | Package (LPA) |
|-----|------|----------------|
| 87  | 8.9  | 6              |
| 111 | 8.9  | 4              |

- **Input (Features)** → `IQ`, `CGPA`
- **Output (Label)** → `Package` (a continuous number, e.g., 6 LPA)

→ This is a **Regression** problem, because the output is a continuous numeric value.

---

## Two Types of Supervised Learning

| Type               | Output              | Example                          |
|---------------------|----------------------|------------------------------------|
| **Regression**       | Continuous number    | Predicting package/salary          |
| **Classification**   | Discrete category     | Predicting placement (Y/N)         |

**Rule of thumb:** if the output can sit on a number line → regression. If the output is a fixed set of categories → classification.

---

## Types of Data (Features)

Data going into a model generally falls into two buckets:

### Numerical
Data that is a measurable quantity.
- Age
- Weight (wt)
- CGPA
- IQ

### Categorical
Data that represents a category/group, not a quantity.
- Gender
- Nation

Knowing whether a feature is numerical or categorical matters because it affects how you preprocess it (e.g., categorical data often needs encoding like one-hot encoding before feeding it into most ML models).

---

## Summary

1. Supervised learning needs **labeled data** — features (X) + a known correct output (y).
2. The model learns the relationship between X and y from historical examples.
3. If y is a **number** → Regression.
4. If y is a **category** → Classification.
5. Features themselves can be **numerical** (age, IQ, CGPA) or **categorical** (gender, nation) — this affects preprocessing.
